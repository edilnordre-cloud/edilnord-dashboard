# Residenza Verdi8

Sito web per Residenza Verdi8 — Via Giuseppe Verdi 8, Brugherio (MB)

## Struttura

```
index.html      → homepage principale
app-a.html      → Appartamento A
app-b.html      → Appartamento B
villa-c.html    → Villa C
attico.html     → Attico
```

## Deploy su GitHub Pages

1. Crea un repository pubblico su GitHub
2. Carica tutti i file da questo ZIP nella root del repository  
3. Vai su Settings → Pages → Source: Deploy from branch → main → / (root)
4. Il sito sarà disponibile su `https://[username].github.io/[repo-name]/`
